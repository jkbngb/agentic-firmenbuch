Connect inside Claude – one click, no API key

For the Claude desktop app (the Chat and Cowork tabs) no API key is required. You add a connector and sign in once by email.

1 Open Settings

Click your name in the bottom left, then click Settings.

2 Add a connector

Open Connectors on the left, click Add at the top right, then click Add custom connector.

3 Enter name & address

Set the name to Agentic-Firmenbuch and the address to https://mcp.agentic-firmenbuch.at/mcp, then choose Add. Leave the OAuth fields empty.

4 Start in a chat – confirm the connection

The connector is now added – there is no "Connect" button here. In a new chat, ask a Firmenbuch question (or activate Agentic-Firmenbuch). On first use Claude asks once to connect, then a browser tab opens for sign-in.

5 Confirm your email

In the browser, enter your email address and choose Send confirmation link. Open the link in the email (valid for 15 minutes) – the connection is established. No API key, no password.

✓ Test it

Start a new chat and ask, for example, "Are you connected to Agentic-Firmenbuch and which tools are available?" If the agent lists the Firmenbuch tools (company search, key figures, history …), the connection is live.
