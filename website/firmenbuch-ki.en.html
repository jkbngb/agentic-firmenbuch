Firmenbuch in Claude

Fastest with Claude Cowork: add the connector https://mcp.agentic-firmenbuch.at/mcp and sign in by email – no API key needed. Or in Claude Desktop / claude.ai under Settings → Connectors → Add custom connector, use the same URL. Then ask, e.g.: “Find active Austrian GmbHs in Upper Austria with a balance-sheet total above €5 million, sorted by revenue.”

Firmenbuch in Cursor

Add an HTTP MCP server to Cursor’s config (~/.cursor/mcp.json) – URL https://mcp.agentic-firmenbuch.at/mcp, header X-API-Key: YOUR_API_KEY (get a free key by email). Then query Austrian company data, ratios and peers right in the editor.

Firmenbuch in VS Code & GitHub Copilot

VS Code and GitHub Copilot (agent mode) support MCP servers. Add the remote MCP https://mcp.agentic-firmenbuch.at/mcp with the X-API-Key header – and run company research right next to your code.

Firmenbuch in ChatGPT

A ChatGPT app for the ChatGPT App Directory is in preparation. Until then, access works through MCP-capable tools like Claude, Cursor, VS Code and GitHub Copilot. Start for free and be notified when the ChatGPT app goes live.

What company data do you get?

  • Master data: legal name, Firmenbuch number, legal form, seat, status.
  • Balance sheet & P&L: filed annual accounts, multi-year, machine-readable.
  • Financial ratios: balance-sheet total, equity ratio, revenue, growth and more.
  • Industry: ÖNACE classification (2025 and 2008).
  • Officers: management / authorised signatories (public Firmenbuch data; birth data year-only).
  • Peers & cohorts: comparison with similar companies, aggregates.
  • Register changes: name, seat, capital and management changes (feed).
