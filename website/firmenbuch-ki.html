Das österreichische Firmenbuch in deinem KI-Tool

Amtliche Firmen- und Bilanzdaten aus dem österreichischen Firmenbuch – direkt in Claude, Cursor, VS Code, GitHub Copilot und jedem MCP-fähigen KI-Assistenten. In natürlicher Sprache nach Firmendaten fragen, geprüfte und quellenbelegte Antworten erhalten – keine Halluzinationen.

Firmenbuch in Claude

Am schnellsten mit Claude Cowork: den Connector https://mcp.agentic-firmenbuch.at/mcp hinzufügen, per Klick mit E-Mail anmelden – kein API-Key nötig. Alternativ in Claude Desktop / claude.ai unter Einstellungen → Konnektoren → Benutzerdefinierten Konnektor hinzufügen dieselbe URL eintragen. Danach im Chat z. B. fragen: „Finde aktive GmbHs in Oberösterreich mit Bilanzsumme über 5 Mio. €, sortiert nach Umsatz."

Firmenbuch in Cursor

In Cursor die MCP-Konfiguration (~/.cursor/mcp.json) um einen HTTP-Server ergänzen – Server-URL https://mcp.agentic-firmenbuch.at/mcp, Header X-API-Key: DEIN_API_KEY (den Key gibt es kostenlos per E-Mail). Dann direkt im Editor österreichische Firmendaten, Kennzahlen und Peers abfragen.

Firmenbuch in VS Code & GitHub Copilot

VS Code und GitHub Copilot (Agent-Mode) unterstützen MCP-Server. Den Remote-MCP https://mcp.agentic-firmenbuch.at/mcp mit dem X-API-Key-Header hinzufügen – und Firmenbuch-Recherchen laufen direkt neben dem Code.

Firmenbuch in ChatGPT

Eine ChatGPT-App für das ChatGPT App Directory ist in Vorbereitung. Bis dahin läuft der Zugang über MCP-fähige Tools wie Claude, Cursor, VS Code und GitHub Copilot. Jetzt kostenlos starten und benachrichtigt werden, sobald die ChatGPT-App live ist.

Welche Firmendaten bekommst du?

  • Stammdaten: Firmenwortlaut, Firmenbuchnummer, Rechtsform, Sitz, Status.
  • Bilanz & GuV: eingereichte Jahresabschlüsse, mehrjährig, maschinenlesbar.
  • Kennzahlen: Bilanzsumme, Eigenkapitalquote, Umsatz, Wachstum u. v. m.
  • Branche: ÖNACE-Klassifikation (2025 und 2008).
  • Organe: Geschäftsführung / Vertretungsbefugte (öffentliche Firmenbuchdaten, Geburtsdaten nur jahresgenau).
  • Peers & Kohorten: Vergleich mit ähnlichen Unternehmen, Aggregate.
  • Register-Änderungen: Namens-, Sitz-, Kapital- und Geschäftsführungswechsel (Feed).
