## 1. sucheFirma (Search Companies)

**Purpose:** Search for companies in the Austrian company register (Firmenbuch)

### Request Parameters

| Parameter | Type | Required | Description | Example |
|-----------|------|----------|-------------|---------|
| `FIRMENWORTLAUT` | string | ✅ | Company name (supports wildcards `*`) | `"maschinenbau*"` |
| `EXAKTESUCHE` | boolean | ✅ | Exact search (true) or phonetic (false) | `false` |
| `SUCHBEREICH` | integer | ✅ | Search scope (see below) | `1` |
| `GERICHT` | string | ❌ | Court code (3 digits) | `"007"` (Handelsgericht Wien) |
| `RECHTSFORM` | string | ❌ | Legal form code (3 chars) | `"GES"` (GmbH) |
| `RECHTSEIGENSCHAFT` | string | ❌ | Legal property | `"Kreditunternehmen"` |
| `ORTNR` | string | ❌ | Location code (1/3/5 digits) | `"1"` (Vienna), `"10707"` (Gols) |

#### SUCHBEREICH Values

| Value | Description |
|-------|-------------|
| `1` | Registered and deleted companies (no branches) |
| `2` | Historical company names (no branches) |
| `3` | No restrictions (all) |
| `4` | Registered company names (exclusivity check) |
| `5` | Companies in draft version |
| `6` | Rejected new registrations |

#### RECHTSFORM Codes (Legal Forms)

> ⚠️ **CORRECTION (verified live 2026-06-19 via `--mode diag`):** the codes originally listed
> here were partly **wrong**. `AKT` and `EGE` return **0 results**. The **verified working
> codes** are: `GES` (GmbH), **`AG`** (Aktiengesellschaft), `KG`, `OG`, `KEG`, `OHG`,
> **`GEN`** (Genossenschaft), `PST` (Privatstiftung), `SE`, `EU` (Einzelunternehmer),
> `SPA` (Sparkasse), `VER` (Versicherungsverein). A `"*"`/under-3-char search also **requires**
> a `RECHTSFORM` or `ORTNR` ("Das Feld RECHTSFORM darf nicht leer sein" / "nur in Kombination
> mit RECHTSFORM oder ORTNR möglich"). `DEFAULT_RECHTSFORMEN` in `enumerate.py` uses the verified set.

| Code | Description | English | Verified |
|------|-------------|---------|----------|
| `GES` | Gesellschaft mit beschränkter Haftung | GmbH | ✅ |
| `AG` | Aktiengesellschaft | Stock Corporation | ✅ (not `AKT`) |
| `KG` / `OG` | Kommandit- / Offene Gesellschaft | Partnerships (current) | ✅ |
| `KEG` / `OHG` | (pre-2007 partnership forms) | legacy partnerships | ✅ |
| `GEN` | Genossenschaft | Cooperative | ✅ (not `EGE`) |
| `PST` | Privatstiftung | Private Foundation | ✅ |
| `SE` | Societas Europaea | European Company | ✅ |
| `EU` | Eingetragener Unternehmer | Sole Proprietor | ✅ |
| `SPA` | Sparkasse | Savings Bank | ✅ |
| `VER` | Versicherungsverein | Insurance Mutual | ✅ |

## Filtering & Best Practices

### 1. Filter by Legal Form (GmbH only)

**Question:** Can I filter to only get GmbH companies?

**Answer:** ✅ Yes! Use `RECHTSFORM="GES"` in `sucheFirma` request.

```xml

```

**Common Legal Form Codes:**
- `GES` = GmbH (Gesellschaft mit beschränkter Haftung)
- `AKT` = AG (Aktiengesellschaft)
- `KEG` = Kommanditerwerbsgesellschaft
- `OHG` = Offene Handelsgesellschaft
- `EGU` = Eingetragenes Einzelunternehmen
